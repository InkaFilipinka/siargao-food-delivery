## How to test

1. Open `index.html` (Design #1), `design-2/index.html` (Design #2), or `design-3/index.html` (Design #3) in Chrome or Safari
2. Choose **Customer** or **Partner**
3. Tap through all screens — navigation is wired (no backend)

## Screens included

**Customer:** Landing → Home → Menu → Cart → Checkout → Track → Orders → Account  
**Partner:** Login → Driver (available + my deliveries) → Restaurant orders → Earnings → Profile

## Usage

1. **Logo** — Use `travelgo-logo.png` for:
   - iOS/Android app icon (1024×1024 for App Store / Play Store)
   - Splash screen
   - Favicon

2. **Design system** — Follow `DESIGN-SYSTEM.md` when building:
   - Customer app
   - Partner app

3. **Preview** — Open `preview.html` in Chrome/Safari to see:
   - Colors and buttons
   - Sample cards (customer home, driver orders)
   - Component styles

## Two Apps

- **TravelGo** (Customer) — Order food, groceries, track delivery
- **TravelGo Partners** (Partner) — Drivers, restaurants, van drivers, guides
